# Supervizor
Project management system

## Endpoints

### Health
<pre>
GET /status/health
</pre>

### Version
<pre>
GET /status/version
</pre>

## Build

### JAR
<pre>
gradlew clean build
</pre>

## Run

### Gradle
<pre>
gradlew clean bootRun
</pre>

### Java
<pre>
java -jar supervizor-1.0-SNAPSHOT.jar
</pre>
